My Project

ผลงานออกแบบเว็บและการเขียนโค๊ด

Weather App

เป็นเว็บที่สามารถดูสภาพอากาศในแต่ละประเทศได้และสามารถดูความชื้นและลมได้

ผมได้เขียนเว็บนี้โดยภาษา HTML Css JavaScript โดยการลิงค์ API จากเว็บไซต์ OpenWeatherMap วีธีการใช้งานคือให้กรอกชื่อเมือง แล้วกด Search หน้าต่างแสดงภูมิอากาศจะแสดงผลออกมา

Quiz App

เป็นเว็บถาม-ตอบ เกี่ยวคำถามในHTMLโดยจะมีคะแนนถ้าตอบถูกคะแนนก็จะเพิ่มตอบผิดคะแนนก็จะลด

ผมพัฒนาเว็บนี้ด้วย HTML Css JavaScript โดยการใช้ startQuiz(); และ resetQuiz(); showScore(); โดยมันจะให้เราตอบคำถามถ้าถูกหมดก็จะรีเซ็ตscore และโชว์ Score

Minecraft Website

ผมได้เขียนรูปแบบเว็บไซต์เป็นเว็บไซต์เซิฟเวอร์เกมมายคราฟโดยเป็นเว็บไซต์เกี่ยวกับเติมเงินภายในเกมชื้อของภายในเกม

พัฒนาโดย HTML และ Css และ การ Link ภายใน Element นั้นๆ

SuperCar Shop

ผมได้ออกแบบเว็บไซต์เป็นโชว์รูมรถยนต์ออนไลน์เกี่ยวกับการโปรโมทรถยนต์และการขายรถยนต์ภายในเว็บ

ผมพัฒนาเว็บไซต์นี้ด้วยการออกแบบโครงสร้างเว็บไซต์โดยใช้ Figma และการเขียนภาษา HTML Css

ผลงานออกแบบเว็บไซต์และกราฟิคต่างๆ

เป็นรูปแบบเว็บเกมออนไลน์มายคราฟสำหรับเติมเงินภายในเกมชื้อของภายในเว็บระบบต่างๆ โดยผมออกด้วยการใช้ Photo shop

ผมได้ออกแบบเว็บไซต์รูปแบบเว็บเกมออนไลน์โดยใช้ PhotoShop เป็นเซิฟเวอร์เกมออนไลน์เกมมายคราฟเกี่ยวกับระบบภายในเซิฟเวอร์ การชื้อของภายในเว็บ

ผมได้ออกแบบลันเซอร์เกม WarZ โดยเป็นเกมเกี่ยวกับฟามของยิงซอมบี้ เป็นเกมแนวเอาชีวิตรอด โดยใช้ PhotoShop ออกแบบ
