About Me

สวัสดีครับผมชื่อ กฤษกร รุจิจนากุล ชื่อเล่น ปีเตอร์ อายุ 18 ปีศึกษาชั้นมัธยมศึกษาปีที่ 6 จากโรงเรียนทวีธาภิเศกแผนการเรียน คณิต-อังกฤษ เกิดวันที่ 18 ธันวาคม พ.ศ 2549

ผมชื่นชอบในการเขียนโปรแกรมมากและอาชีพในฝันของผมคือWeb Developerและนี่ก็คือ Portfolio รวบรวม Project ผลงานของผมทั้งหมดครับ

The reason it was made

สาเหตุที่ผมทำเว็บไซต์ Portfolio ขึ้นมานั้นผมได้ลองทดสอบทักษะการเขียนโปรแกรมที่ศึกษาเรียนรู้อยู่มาต่อยอดในการเขียนโปรแกรมลงในนี้เพื่อเพิ่มความสามารถของผม โดยใช้ ภาษา HTML และ Css ผมได้ลองใช้ Animation เพื่อเพิ่มลูกเล่นส่วน Framwork ที่ผมกำลังศึกษาอยู่นั้น คือ Tailwind Css และ Next.js และ React.js ภาษาที่ผมกำลังศึกษาเพิ่มเติมอยู่คือ Java Script ประสบการณ์ที่ได้รับจาการทำโปรเจคน์นี้คือสอนให้ผมแก้ปัญหาต่างๆภายในโค๊ด และได้เรียนรู้เทคนิคใหม่ๆ
